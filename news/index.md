## epiwraps 0.99.113

### Significant changes

## epiwraps 0.99.80

### Significant changes

- Version \>0.99.80 introduced some major changes, in particular the
  introduction of `EnrichmentSE` objects, which are now the default
  output of `signal2Matrix` and facilitate manipulations, as well as
  renaming of some normalization functions under a more standardized
  interface.
